# Project Roadmap

## Phase 1 — Business Understanding

### Tasks

- Search for a suitable real-world dataset.
- Understand the business domain.
- Define the business problem.
- Identify stakeholders.
- Determine the target variable.
- Define the machine learning problem type:
  - Classification
  - Regression
  - Clustering
- Define success metrics.

---

## Phase 2 — Data Understanding

### Tasks

- Load the dataset.
- Inspect the dataset.

### Check

- Dataset shape
- Columns
- Data types
- Missing values
- Duplicate values
- Unique values
- Statistical summary
- Class distribution
- Initial observations

---

## Phase 3 — Data Cleaning

### Tasks

- Handle missing values.
- Remove duplicates.
- Fix incorrect data types.
- Detect and treat outliers.
- Clean inconsistent values.
- Rename columns if necessary.

---

## Phase 4 — Exploratory Data Analysis (EDA)

### Univariate Analysis

- Numerical distributions
- Categorical distributions
- Datetime analysis (if available)

### Bivariate Analysis

- Feature vs Feature
- Feature vs Target

### Multivariate Analysis

- Correlation Heatmap
- Pairplot
- PCA (Optional)

---

## Phase 5 — Feature Engineering

### Tasks

- Encode categorical variables.
- Scale numerical features.
- Create new features.
- Perform feature selection.
- Handle class imbalance (if needed).

---

## Phase 6 — Modeling

### Tasks

- Split the dataset.
- Train at least two machine learning models.
- Compare model performance.
- Tune hyperparameters (optional but recommended).

---

## Phase 7 — Evaluation

### Classification Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
- Confusion Matrix

### Regression Metrics

- MAE
- MSE
- RMSE
- R² Score

---

## Phase 8 — Deployment

Build an interactive web application using **Streamlit**.

### Features

- Upload or enter input data.
- Make predictions.
- Display prediction results.
- Show basic project information.
- Display selected EDA visualizations (Optional).
